# ucfirst

Capitalizing the first character in a string is done in Ruby using the
`String#capitalize` method.

{{code:php
    print ucfirst("hello world");
    // => Hello world
}}

{{code:ruby
    puts "hello world".capitalize
    # => Hello world
}}
