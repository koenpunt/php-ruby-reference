# strtoupper

Making an entire string uppercase is done using Ruby's `String#upcase` method.

{{code:php
    print strtoupper('Hey World');
    // => HEY WORLD
}}

{{code:ruby
    puts "Hey World".upcase
    # => HEY WORLD
}}
